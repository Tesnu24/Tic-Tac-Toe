
# 🎮 Tic Tac Toe Game

A simple Tic Tac Toe game built using HTML, CSS, and JavaScript.

## Features
- Single Player (vs Computer)
- Two Player Mode
- Player name input
- Responsive design

## How to Play
Choose a mode, enter names, and take turns marking X or O. First to get 3 in a row wins!

## Tech Used
HTML, CSS, JavaScript
